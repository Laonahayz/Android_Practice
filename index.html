📘 ShopDuDu 商品列表 App 教學進度

🔹 開發環境建置

項目說明
安裝 Android Studio使用官方最新版，含 SDK / 模擬器安裝
建立 Kotlin App 專案使用 Empty Activity，命名為 ShopDuDu
模擬器管理建立多台模擬器（Pixel 4a / Pixel 5 / Pixel Tablet）以測試不同尺寸畫面

🔹 Gradle 設定與版本管理

項目說明
使用 Kotlin DSL（.kts）Gradle 使用 Kotlin 語法格式
套件管理使用 Version Catalog經由 libs.versions.toml 統一集中管理套件版本
設定 DataBinding在 buildFeatures 中啟用 dataBinding = true

🔹 加入的核心套件（Library）

套件功能說明
retrofit串接 RESTful API（網路請求）
converter.gson將 JSON 資料自動轉為 Kotlin 資料類別
coroutines.android使用協程處理非同步請求（不卡 UI）
lifecycle.viewmodel.ktx資料邏輯與畫面分離的 ViewModel
lifecycle.livedata.ktx觀察資料變化並更新畫面的 LiveData
recyclerview建立滑動式商品列表（可重複顯示項目）

🔹 資料模型建立

類別名稱用途欄位
Product.kt 對應 API 回傳的商品資料 id, title, price, description, image

  • 基礎套件與 Gradle 設定完成
  • 商品資料模型 Product.kt 建立完成
